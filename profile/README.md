🧙 [github上的sysu](https://github.com/sysu)是github最早的中大人的技术成果开放地之一、中大人自主治理与管控的人数较多的开源开放地之一。是无学院属性的综合内容的开源开放地、兼顾实验性与工程价值的项目归档目录，立足具体项目的动脑与动手，辐射历史遗产项目保护、信息科学与技术、人工智能、管理学、跨学科等现实领域，多层次融入社会科技生命周期。是汇编级的资料库、源代码级的项目库、中大人的小花园、中大人的咖啡馆。是开源的根目录、中大人的历史记忆。是热心师生校友的计划交流平台、技术交流平台、自律互助平台、项目管理平台、协作互动平台和综合交流平台。

👋 欢迎将年轻的和经验丰富的师生校友[添加](https://github.com/orgs/sysu/people)进来。请联系您的学长学姐或有相关操作权限的人士。

🍿 生活愉快！ Have Fun！
